# Credits

City Builder Survival uses the following third-party 3D assets. All models are
distributed under permissive licenses (CC0 unless otherwise noted).
Textures have been embedded into each `.glb` file; contents are otherwise unmodified
(except where noted: a few GLBs combine two or more models into one scene, and
character GLBs keep only the gameplay animation clips, renamed for the game's
clip lookup).

The game uses a contemporary set: Kenney city kits for the modern backbone,
KayKit for props/tech, Quaternius for characters and several specialty models.
